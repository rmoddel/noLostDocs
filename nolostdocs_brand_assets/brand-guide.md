# NoLostDocs Brand Starter Kit

## Core Promise
**No more lost docs. Everything you need in one app.**

## Brand Personality
- Secure, but not scary
- Clean, but not cold
- Practical, not gimmicky
- Consumer-friendly, but serious enough for sensitive documents

## Logo Concept
Rounded app tile + document stack + folder + checkmark.

## Color Palette
- Midnight Navy `#102033`
- Trust Blue `#2F80ED`
- Safe Mint `#19B394`
- Soft Cloud `#F7FAFC`
- Slate `#526173`
- Line `#DCE5F2`
- White `#FFFFFF`

## Font Direction
Use Inter, Manrope, or SF Pro style sans-serif.
