## Copy these into Next.js public

```bash
mkdir -p public/icons public/logos
cp unfurl/og-image.png ../public/og-image.png
cp unfurl/twitter-card.png ../public/twitter-card.png
cp icons/* ../public/icons/
cp logos/* ../public/logos/
cp seo/site.webmanifest ../public/site.webmanifest
cp seo/robots.txt ../public/robots.txt
cp seo/sitemap.xml ../public/sitemap.xml
```

## Homepage crawl requirement

Make sure the initial HTML includes:

```txt
No more lost docs.
Everything you need in one app.
Securely store, organize, and find your important documents.
```
