# Sketch 002: Launcher Preview Color System

## Design Question
What palette should carry the Launcher Preview concept so NoLostDocs feels premium, trustworthy, and modern without reading like a luxury wallet brand or a fintech startup?

## How to View
open .planning/sketches/002-launcher-preview-color-system/index.html

## Variants
- **A: Heritage Vault** — warm bone, espresso, and brushed-gold accents; closest to the leather-inspired direction but cooled for document trust.
- **B: Soft Carbon** — dark graphite and restrained electric blue; more technical and app-like without going full crypto neon.
- **C: Editorial Copper** — off-white, slate, and oxblood-copper; stronger editorial personality and warmer contrast.

## Selected Direction
**A/C hybrid** — primarily Heritage Vault for trust and calm, with a measured amount of Editorial Copper character to avoid feeling too plain.

## What to Look For
- Which palette feels most trustworthy for sensitive personal records.
- Which palette still feels modern on mobile without becoming generic fintech.
- Whether the accent color is premium and controlled instead of loud.
