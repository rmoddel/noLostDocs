# Sketch 001: Locked Preview Homepage

## Design Question
How should NoLostDocs present the app-first category dashboard to anonymous users without pretending they are already inside the authenticated product?

## How to View
open .planning/sketches/001-locked-preview-homepage/index.html

## Variants
- **A: Launcher Preview** — the homepage behaves like a premium locked app launcher with large category tiles and a bottom login tray. ★ Selected
- **B: Guided Preview** — the homepage explains the flow more directly with staged preview cards and a stronger sign-in narrative.
- **C: Split Trust + Preview** — the homepage balances trust/auth copy on top with a stronger app preview panel below.

## What to Look For
- Which variant feels closest to “the app is the product” without confusing anonymous users.
- Whether login feels like a natural gate instead of a jarring interruption.
- Whether the category system is understandable at a glance on mobile.
