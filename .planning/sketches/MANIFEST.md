# Sketch Manifest

## Design Direction
Mobile-first, trust-first entry into NoLostDocs. Anonymous users should understand the category-first app model immediately, but real document state and personalization stay behind login. The homepage behaves like a locked preview of the real app rather than a traditional marketing splash.

## Reference Points
- Mobile launcher-style dashboards
- Locked preview onboarding
- Trust-forward auth entry for secure utilities
